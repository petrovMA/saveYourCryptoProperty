Warning! Use only Tezos Alpha-net

1. Deploy a smart contract on the Tezos network. file GitHub (This step will be more simple in the near future, via "deploy" button)

2. Enter the new contract address in the "Contract address" field

3. Now you can change the contract parameters using this methods:
"Your Balance" - show balance (method for "recipient" and "owner" accounts)
"Ping" - set the current time for ping time (method for "owner" account).
"Receive" - Recipient can get properties (all ꜩ from contract) when the difference between the current time and the ping time is greater than the delay time (method for "recipient" account).
"Set new owner" - change the owner of the smart contract property (method for "owner" account).
"Change delay" - Set a new delay time (method for "owner" account).
"Withdraw" - Withdrawal of some ꜩ from the smart contract (method for the owner).(method for "owner" account).
"Set recipient" - Set a new recipient (method for "owner" account).
"Delegate" - Delegate smart-contract ꜩ tokens (method for "owner" account). [In development].
